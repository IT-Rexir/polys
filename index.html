About Polyskimchis SMP

Polys Kimchis SMP is a server that supports crossplay for both Java and Bedrock editions, offering a community-focused experience with an active economy and dedicated shopping districts. It operates as a 24/7 hosted server with a peaceful playstyle, meaning griefing, stealing, and non-consensual PvP are strictly prohibited to maintain a mature environment.
